# HTML Practice Files

This repository contains three basic HTML examples to demonstrate different elements of web page creation.

## Files

### 1. list.html
- Contains paragraphs, headings, links, and an unordered list.
- Demonstrates:
  - `<p>` for paragraphs
  - `<h1>` for headings
  - `<a>` for hyperlinks
  - `<ul>` and `<li>` for lists

### 2. NestedList.html
- Shows how to create nested lists.
- Demonstrates:
  - `<ul>` for unordered lists
  - `<li>` for list items
  - Nested structure for categories (Fruits, Vegetables, Cereals, Nuts)

### 3. profile.html
- A simple profile page for "Pochi the Cat."
- Demonstrates:
  - `<h1>` and `<h2>` for headings
  - `<p>` for text
  - `<ol>` for ordered lists
  - `<img>` for images
  - `<a>` for links
  - `<hr>` for horizontal rule

## Purpose
These files are intended for learning and practicing basic HTML concepts such as:
- Structuring text with headings and paragraphs
- Creating lists (ordered, unordered, nested)
- Adding links and images
- Using simple formatting elements

## How to Use
1. Open any of the `.html` files in a web browser.
2. Observe how the HTML elements are rendered.
3. Modify the code to experiment with different tags and attributes.
