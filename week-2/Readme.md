## Key Concepts

### Shallow Copy
Creates a copy of an object, but nested objects still refer to the original memory.

### Deep Copy
Creates a completely independent copy, including nested objects.

## How to Run

1. Install Node.js on your system  

2. Clone the repository  

```
git clone <your-repository-url>
```

3. Navigate to the project folder  

```
cd week-2/system-management
```

4. Run any file  

```
node filename.js
```

Example:

```
node examSimulator.js
```
